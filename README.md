## Dioxide Music Bot
music bot written in dcjs v13 node.js 

## features
- 40 total commands
- 35+ music commands
- dev commands
- utility commands

## Setup
- download dioxidex47jsk.zip
- open /source
- use index.js as main file
- use node v20 to avoid errors
- enter your token and other details in config.js
